# Ternary Operator

- can be used to replaced JS if...else statements
- Syntax: ( ? : )

### Example

- if...else statement

```js
let isCorrect = true;

if (isCorrect) {
  console.log("Correct!");
} else {
  console.log("Incorrect!");
}
//Prints: Correct
```

<br>
<br>

```js
let isCorrect = true;

isCorrect ? console.log("Correct!") : console.log("Incorrect!");
//Prints: Correct
```
